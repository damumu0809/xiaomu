## Ubuntu 下安装 fcitx 拼音输入法

### 1. 添加 fcitx 源

首先在终端输入：

```
sudo add-apt-repository ppa:fcitx-team/nightly
```

或

```
sudo add-apt-repository ppa:fcitx-team/stable
```

然后更新软件源：

```
sudo apt-get update
```


### 2. 安装

```
sudo apt-get install fcitx-pinyin 
```
